# 💰 Expense Tracker

A beautiful Expense Tracker web application built using HTML, CSS, Bootstrap, and JavaScript.

---

## 🚀 Features

- ➕ Add Expenses
- 🗑 Delete Expenses
- 📂 Category Selection
- 💾 Data stored using LocalStorage
- 📊 Total Expense Calculation
- 🎨 Responsive UI Design using Bootstrap

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- LocalStorage

---

## 📸 Project Preview

This project helps users track daily expenses like:

- Food 🍔
- Travel ✈️
- Shopping 🛒
- Petrol ⛽
- Others 📌
